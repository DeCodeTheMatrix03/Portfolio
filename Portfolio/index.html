About Me

A Software Engineer based in NYC with a deep interest in working with software that improves the world. Before programming, I taught Math to 5th and 7th-grade students. I realized technology was the focal point of all classrooms and decided to pursue my passion for tech. It was during this time I challenged myself to create static websites using HTML and CSS.

Later moving on to using the MERN Stack to create full-stack web applications and discovered my love for software engineering. My experience in education coupled with my programming skills gives me a unique perspective that I would love to bring to a team!

Find Your Routine

This is an online web application that allows users to explore exercises for specific muscle groups.I built this application with the thought of helping someone fulfill their fitness goals by simplifying the process of finding a workout routine! A React Web Application that utlizes RapidAPI to pull workouts from a Workout API. The frontend is comprised of React.js and MaterialUI for styling and composing.

Tech Used: React, CSS, MaterialUI

Wellness Journal

A FullStack application that utilizes Model, View, Controller structure to help users keep track of their fitness goals. This app allows users to document their workouts, nutrition, and sleep to help optimize performance and recover. My goal was to allow user’s one place to document all aspects of their fitness journey.

Tech Used: React, Node.js, MongoDB, Mongoose, Express

Online Bookstore

Tech Used: HTML, CSS, Stripe
